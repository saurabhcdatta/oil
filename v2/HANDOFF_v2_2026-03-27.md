### yyyyqq Parsing — BULLETPROOF VERSION
```r
parse_q <- function(x) {
  x <- as.character(x)
  result <- rep(as.Date(NA), length(x))
  valid  <- grepl("^[0-9]{4}\\.[1-4]$", x)
  if (any(valid)) {
    p             <- str_split_fixed(x[valid], "\\.", 2)
    yr            <- as.integer(p[, 1])
    qtr           <- as.integer(p[, 2])
    mo            <- (qtr - 1L) * 3L + 1L
    result[valid] <- as.Date(sprintf("%04d-%02d-01", yr, mo))
  }
  result
}
```

## PART 4: R PITFALLS (ALL HARD-WON)
================================================================================

1. **coef(summary(feols)) returns EMPTY MATRIX**
   ALWAYS use: coef(fit), se(fit), pvalue(fit) — fixest native API
   Or: summary(fit)$coeftable as fallback

2. **vcovCL(cluster=~join_number) FAILS when fixest loaded**
   round_any dispatches to fixest method — use plain integer vector:
   ```r
   cl_vec <- as.integer(factor(df$join_number))
   sandwich::vcovCL(fit_lm, cluster = cl_vec)
   ```

3. **feols drops time-only-varying regressors**
   Macro vars (same for all CUs in a quarter) get absorbed by quarter FE
   AND dropped by CU FE. Use lm() + sandwich for macro var regressions.

4. **haven_labelled breaks as.double()**
   Always zap_labels() and zap_formats() before numeric coercion.

5. **xgboost: evals= not watchlist=** (changed in xgboost >= 1.7)

6. **yyyyqq parsing fails on NA/empty**
   Validate with grepl("^[0-9]{4}\\.[1-4]$") before any split/conversion.

7. **rm(list=ls()) before every source()**
   Stale fixest objects in session cause round_any errors that look
   unrelated to fixest. Always source atomically.

8. **for(pnl_obj in list(...)) copies data.tables**
   Return new DT from helper function instead.

9. **abs(x, na.rm=TRUE) is invalid**
   na.rm belongs in max()/sum(), not abs().

10. **%% double-percent in non-sprintf strings**
    Use single % in regular strings; %% only inside sprintf format strings.

11. **fct_rev() requires forcats loaded**
    Use base R: factor(x, levels=rev(levels(x)))

12. **Single-column matrix in xgb.DMatrix**
    Use X[rows, , drop=FALSE] and storage.mode(X) <- "double"

13. **get_fixest_cf() helper pattern (use this everywhere)**
    ```r
    get_fixest_cf <- function(fit) {
      tryCatch({
        b <- coef(fit); se <- se(fit); pv <- pvalue(fit)
        rn <- gsub("`", "", names(b))
        data.frame(Estimate=as.numeric(b), `Std. Error`=as.numeric(se),
                   `Pr(>|t|)`=as.numeric(pv), row.names=rn, check.names=FALSE)
      }, error=function(e) {
        tryCatch({
          ct <- summary(fit)$coeftable
          rn <- gsub("`", "", rownames(ct))
          data.frame(Estimate=ct[,1], `Std. Error`=ct[,2], `Pr(>|t|)`=ct[,4],
                     row.names=rn, check.names=FALSE)
        }, error=function(e2) data.frame())
      })
    }
    ```

## PART 6: METHODOLOGY REFERENCE
================================================================================

### Identification Strategy (Bartik Shift-Share)
- Instrument: Z_it = dlog_oil_t × oil_exp_s(i)
- where oil_exp_s(i) = pre-period (2000-2010) NAICS 211 share of state s
- Variation: cross-sectional (oil-state vs non-oil-state) not just time-series
- Exclusion restriction: pre-period exposure is uncorrelated with current shocks
- Reference: Goldsmith-Pinkham et al. (2020 AER)

### Mediation Framework (Baron-Kenny)
- Link 1: oil → macro (time-series OLS, NOT feols)
- Link 2: macro → CU (lm() + vcovCL with plain integer cluster vector)
- Link 3: AR persistence (feols with CU FE only, no quarter FE)
- Indirect effect: a × b (Sobel SE = sqrt(b²*se_a² + a²*se_b²))
- Direct effect: c' = c - Σ(a×b)

### VARX Specification (Script 04b)
- Cholesky ordering: cost_of_funds → dq_rate → netintmrg →
                     loan_to_share → pcanetworth → insured_share_growth
- Exogenous: oil price, macro CCAR variables
- Lags: P=2 (AIC-selected)
- IRF horizon: 8 quarters with 90% bootstrap CI (green=positive, red=negative)
- FEVD: at 8Q horizon

### XGBoost Specification (Scripts 04c, 04e)
- 72-combo grid: eta(3) × max_depth(3) × subsample(2) × colsample(2) × min_child(2)
- evals= (not watchlist=) for xgboost >= 1.7
- val_rmse extracted from evaluation_log with grep("_rmse$") fallback
- TreeSHAP: predict(fit, data, predcontrib=TRUE)
- PATHWAY_LABELS safe codes: D, INT, LAB, CPI, RTE, HPI, BAL, OTH

### Kilian SVAR (Script 06)
- 3-variable: [global_prod, REA_proxy, real_oil_price]
- Proxy spec (no EIA data): [GDP_growth, real_oil_price] (2-variable)
- Cholesky ordering: supply most exogenous, price most endogenous
- Structural shocks: supply=col1, demand=col2, geo=col3
- Iran war = geopolitical/precautionary type (column 3)
